# nice!view glyphs shield

This shield variant provides glyph-based pattern image rendering via configurable glyph layouts.

To fall back to the built-in ZMK status screen instead of the custom glyphs widget:

```
CONFIG_ZMK_DISPLAY_STATUS_SCREEN_BUILT_IN=y
CONFIG_ZMK_LV_FONT_DEFAULT_SMALL_MONTSERRAT_26=y
CONFIG_LV_FONT_DEFAULT_MONTSERRAT_26=y
```
